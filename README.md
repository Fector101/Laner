# Laner

A versatile app enabling file browsing between PC and phone seamlessly.

## Features

- **View PC Files on Phone**: Effortlessly access PC directories from your phone.
- **View Phone Files on PC**: Seamlessly browse phone storage on your PC.

## Technologies Used

- **Python**: Backend logic and Kivy for UI.
- **Buildozer**: Packaging for mobile platforms.

## Getting Started

1. Clone this repository:

```bash
git clone https://github.com/Fector101/Laner.git
```

2. Follow the setup instructions in `buildozer.spec` for packaging.
